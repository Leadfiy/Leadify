# 6. Admin Portal

![Admin Start](../assets/images/Admin_Start.png)


Dies ist das Startmenü, das nach der Anmeldung mit der Administratorrolle angezeigt wird. 
Im oberen Menü befindet sich links das Hamburger-Menü. Dieses dient als Schnellzugriff auf die beiden Hauptmenüpunkte. Daneben wird die aktuelle Ansicht angezeigt (hier: „Administrator“).
Ganz rechts befindet sich das Benutzermenü. Hier können das Konto verwaltet, sich abgemeldet oder zwischen Light- und Dark-Mode gewechselt werden.
Links daneben befindet sich das Nachrichtenfenster, welches anzeigt, wie viele neue Meldungen oder Aufgaben vorhanden sind.
Darunter befinden sich zwei Schaltflächen:

•	Leads löschen: Zeigt an, wie viele Leads zur Löschung vorgemerkt sind. 
•	Nutzerfreigabe: Hier werden neue Mitarbeiter zur Freigabe angezeigt.

![Lead Löschen Oberfläche](../assets/images/Lead_Löschen_Oberfläche.png)


In dieser Ansicht befindet sich die Oberfläche zum Löschen von Leads.

Links oben kann über ein Dropdown-Menü nach der Art gefiltert werden.
Rechts daneben befindet sich die Schaltfläche zur Bestätigung der Löschung aller ausgewählten Einträge.
Unterhalb des Dropdown-Menüs gibt es die Möglichkeit, alle vorgemerkten Einträge auszuwählen.

Darunter wird eine Liste aller zur Löschung vorgemerkten Aufträge angezeigt, die per Mausrad durchsucht werden kann.

![Lead endgültig löschen](../assets/images/Lead_endgültig_löschen.png)

Beim endgültigen Löschen öffnet sich ein Bestätigungs-Popup. Dieses dient dazu, versehentliche Löschungen zu vermeiden.
Nach Bestätigung über die rote Schaltfläche „Endgültig löschen“ wird der Vorgang abgeschlossen.


![Nutzer Freigabe](../assets/images/Nutzer_Freigabe.png)

Diese Ansicht zeigt die Nutzerfreigabe.
Hier wird ein neuer Benutzer angezeigt, dessen Daten geprüft werden müssen. Anschließend kann der Nutzer:

•	über die grüne Schaltfläche freigegeben oder 
•	über die rote Schaltfläche abgelehnt werden.
